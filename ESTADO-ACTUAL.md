## 🚀 Próximos Pasos

### Para cambiar la imagen principal:
1. Reemplaza `images/hero-image.jpg` con tu nueva imagen
2. Mantén el mismo nombre de archivo
3. Sube a GitHub:
   ```powershell
   git add images/hero-image.jpg
   git commit -m "Update hero image"
   git push origin main
   ```
